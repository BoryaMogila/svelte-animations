VinnytsiaJS 2020

П'ята щорічна Javascript конференція, яка відбудеться у Open-Air форматі. Захід організовує Vinnytsia JavaScript Community. Цього літа зустріньте Конференцію, присвячену JavaScript розробці. Ми подбали про все, щоб забезпечити усім гостям конференції комфортну локацію, щоб ви просто відпочивали та насолоджувались спілкуванням.
